Partie 1 : Installation (La "Valise")
Créez un dossier tp_node et ouvrez votre terminal dedans.

Initialisation : npm init -y

Installation des outils (Extensions 2, 4, 6) :

Bash

npm install express mongoose socket.io multer cors
express : Framework web (remplace le code lourd de routing).

mongoose : Pour parler à MongoDB.

socket.io : Pour le temps réel.

multer : Pour gérer l'upload d'images.

Partie 2 : Le Code "All-in-One" (Extensions 1, 2, 4, 6)
Plutôt que de faire 50 fichiers, voici une implémentation robuste qui couvre le routage (Express), la base de données (Mongoose), l'upload, et la logique métier.
